
# PizzaOven — интернет-магазин пиццерии

**PizzaOven** — это полноценный веб-сайт для заказа пиццы онлайн. Проект реализован на Django и включает в себя каталог товаров, корзину, оформление заказов и панель администратора. Разработан в качестве дипломной работы. 

## Возможности

-  **Каталог** с категориями и подробным описанием
-  **Корзина** — добавление, изменение количества, удаление товаров
-  **Оформление заказа** с формой контактных данных
-  **Регистрация и авторизация** пользователей
-  **Панель администратора** для управления товарами, заказами и пользователями

## Технологии

- Python 
- Django 
- HTML5 / CSS3
- JS

### Установка и запуск (Windows)

````
git clone https://github.com/FlanRok/PizzaOven.git
cd PizzaOven
py -m venv venv
venv/scripts/activate
pip install -r requirements.txt
cd pizzaoven
python manage.py runserver
````
